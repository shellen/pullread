What's New

Release notes for Pull Read, from first light to the latest build.

0.4.8 "The Reading Room"
April 2026
  • Research tab — a knowledge graph of the people, companies, places, and topics that show up across your reading, with an interactive canvas and entity detail view
  • Daily email rundown — subscribe to a daily mini-magazine of your unread articles with an AI-written editorial summary, delivered to any SMTP address
  • Notebooks feed the graph — notes you write now extract into the same knowledge graph as articles, so your own thinking connects to its sources
  • New Claude models — Opus 4.7 and Sonnet 4.6 are selectable under Settings › LLM
  • Viewer you can always find again — click the menu-bar icon or the Dock icon to bring the window back, with desktop notifications when a sync finishes
  • Podcast & duplicate fixes — podcast episodes now prefer feed content over webpage extraction, NPR embeds no longer leak as visible text, and leftover duplicates from folder migrations are cleaned up
0.4.7 "The Welcome Mat"
March 2026
  • Fixed a bug where new users saw “PullRead Server Error” on first launch instead of the onboarding experience
  • Fixed feed subscription during onboarding — the “Subscribe” button now works reliably on first run
  • Better error messages when the viewer server encounters issues
0.4.6 "The Social Butterfly"
March 2026
  • Social post cards — Bluesky, Mastodon, and X posts now render as native-looking cards with avatars, timestamps, and platform branding
  • Engagement metrics — see reply, repost, like, and quote counts on Bluesky posts with a link to interact on the original
  • Quoted posts — Bluesky quoted posts are embedded inline within the card
  • Social sources — new “Social” section in the Sources sidebar to browse all your social bookmarks
  • Smarter Rundown briefings — article links are more reliable across all LLM providers including Apple Intelligence
  • Briefing loading indicator — “Generating your briefing…” status text while the AI works
  • LLM attribution — hover the refresh button on the Rundown to see which model generated the briefing
0.4.5 "The Defuddler"
March 2026
  • Fixed a crash on launch caused by a build-time dependency (jsdom) that broke in compiled binaries — replaced with a lightweight alternative
  • Fixed feeds with image thumbnails (like BrooklynVegan) showing only the RSS summary instead of the full article
  • Fixed duplicate articles appearing in the dashboard and sidebar when the same file existed in multiple locations
  • Added a CI safeguard to prevent the jsdom crash from recurring in future builds
0.4.4 "The Reconnect"
March 2026
  • Fixed “View Articles” opening a blank localhost page when the viewer server failed to start
  • Viewer server now automatically retries on startup failure instead of opening a dead URL
  • Fixed a bug where a crashed viewer server could never restart without quitting the app
  • Server liveness check — PullRead now verifies the viewer is actually responding before reusing a cached port
0.4.3 "The sidebar shuffle"
March 2026
  • Flat sidebar navigation — Explore, All Items, Sources, Starred, and Notebook are all top-level nav items
  • Sources & Tags drawer — tags now live inside the Sources drawer instead of a separate nav item
  • Explore page — renamed from Home, serves as the card-based landing view
  • Better card images — og:image extraction and smarter fallbacks for story deck cards
  • CLI documentation — in-app Guide and FAQ now cover Terminal setup and CLI usage
  • Default to Light theme and Instrument Serif font on first launch
  • Beta features toggle — opt in to experimental features like Ask from Settings > Advanced
  • Feedback button — now shows a modal with copyable support address
  • Better feed subscription UX — inline validation errors, shortcut hints, duplicate detection, and success feedback
  • Feed shortcuts — paste r/subreddit, @user.bsky.social, or example.substack.com and PullRead finds the feed
  • Feed picker shows “all caught up” when you’re already subscribed to all suggested feeds
  • Bluesky profile URL support for feed subscriptions
  • OpenAI GPT-4.1 and GPT-5.x model support
  • Notebook actions moved to reader toolbar with overflow menu
  • Fixed Safari text overlay positioning on card carousels
0.4.2 "The Rundown"
March 2026
  • Rundown tab — unified home view replaces the separate Brief, Sections, and Spotlight tabs
  • Story deck — swipeable vertical card showcasing top unread articles with images
  • AI briefing — concise overview of your reading queue with clickable article links
  • Trending topics — briefing highlights categories with the most new articles
  • Continue Reading card — pick up where you left off with progress indicators
  • Latest card — newest unread articles at a glance
  • Section cards — editorial categories with article counts and featured thumbnails
  • No-AI fallback — reading suggestions shown when AI is unavailable
  • Instagram embed rendering fix
0.4.1 "Play everything"
March 2026
  • Video podcasts — play video enclosures and HLS streams inline or in a popout
  • YouTube playback — embedded player with queue integration and popout window
  • Editorial sections — articles auto-classified into 12 sections like Tech, Science, and Culture
  • For You redesign — section-grouped rundown with proportional slot allocation
  • Magic Mixer — fine-tune Magic sort ranking with per-topic sliders
  • Bookmarks source — imported bookmarks and inbox saves grouped in the sidebar
  • Auto-tag on sync — new setting to automatically tag articles when feeds sync
  • Apple Intelligence performance — pre-compiled Swift binaries eliminate per-article compilation overhead
  • Batch autotagging — all articles processed in a single binary invocation
  • Comment threads — view and reply to article comments
  • Offline mode — all CDN dependencies bundled for fully offline operation
  • Storage management — see disk usage and clean up old articles from Settings
  • Clicking a source name in article headers opens the source drawer
  • Fixed Apple Intelligence context window errors on long articles
  • Fixed YouTube popout playback issues (error 153, auto-advance, DOM construction)
